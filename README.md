# Weather App 🌤️

This is a simple weather app built using HTML, CSS, and JavaScript.

## Features

* Search weather by city name
* Shows temperature, weather condition, and location
* Simple and clean UI

##  Tech Used

* HTML
* CSS
* JavaScript

## How to Use

1. Enter city name
2. Click search
3. Get weather details instantly

## Note

This project is created for learning and practicing JavaScript and API integration.
